### Backend
```sh
$ cd Backend
$ dotnet restore
```

### Frontend

```sh
$ cd Frontend
$ npm i react-scripts
```

## Run

```sh
$ cd Backend
$ dotnet run
```
new terminal window

```sh
$ cd Frontend
$ npm install
$ npm start
```

react server at `localhost:3000` 

backend apis are at `localhost:8000/api`

swagger api docs at `localhost:8000/swagger`
